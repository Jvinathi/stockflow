# StockFlow

Multi-tenant SaaS inventory and order management system built with FastAPI, React, and MySQL.

> Full README with setup instructions, architecture overview, and screenshots coming in the next update.

## Tech Stack
- Backend: FastAPI, SQLAlchemy, MySQL, JWT, APScheduler, WeasyPrint
- Frontend: React (Vite), Tailwind CSS, Recharts, Axios

## Status
Actively in development — core features complete (multi-tenancy, RBAC, inventory, orders, invoicing, low-stock alerts, analytics dashboard).
